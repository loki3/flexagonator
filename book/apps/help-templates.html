Help: Flexagon Templates

The Flexagon Templates page is a companion app for the book The Secret World of Flexagons.

Select figure: Select the figure number for the template you would like to use from the list in the upper left corner.

Printing and folding: See Help: Making Flexagons for tips on printing and folding the templates made by Flexagon Templates.

Providing links: If you want to send someone a link to a specific figure, add the figure number to the templates URL. For example, the link to figure 29.9 is
https://loki3.github.io/flex/templates.html?figure=29.9.

For more information, see the book The Secret World of Flexagons.
