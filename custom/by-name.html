Flexagon Naming

This is intended to provide some examples of how flexagon names map to actual flexagons. It is not intended to be an exhaustive list of different flexagons. Note that many of these names are ambiguous, so flexagonator may have chosen just one possible representation. This list is focused on flat triangle flexagons, leaving out non-flat ones (some with twists) or ones with other leaf shapes (quadrilateral, pentagon, etc.).
